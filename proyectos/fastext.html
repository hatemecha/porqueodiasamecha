Descripción

Editor de texto minimalista para escritorio construido con Tauri y Rust. Diseñado para ser simple, rápido y eficiente.

Características

- Múltiples pestañas para trabajar en varios documentos a la vez.
- Temas personalizables.
- Soporte para Markdown.
- Interfaz limpia y sin distracciones.

Tecnologías

Backend: Rust + Tauri 1.5 | Frontend: HTML, CSS, JavaScript
